### Главный сценарий демо

Показываем одну простую историю:

1. Claude Code / Codex упёрся в лимит
2. задача ещё не закончена
3. открываем ChatGPT
4. GPT через Second Lane вызывает `inspectProject`
5. GPT читает файл и вносит правку
6. GPT запускает тест или команду
7. видно успешный результат

### Что должен понять зритель

- это не просто чат;
- ChatGPT реально работает с проектом;
- Second Lane нужен именно в момент, когда другой агент остановился.
