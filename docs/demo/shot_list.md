# Second Lane Demo Shot List

## RU

### Shot 1 — Pain

- короткий текст на экране:
  - `Claude Code hit a limit before the task was done`
  - или `Codex stopped but the task still had to ship`

### Shot 2 — Move into ChatGPT

- открыт ChatGPT
- выбран Custom GPT
- видно, что работа продолжается

### Shot 3 — Inspect

- GPT вызывает `inspectProject`
- виден ответ по проекту

### Shot 4 — Read / Patch

- GPT читает нужный файл
- GPT делает правку

### Shot 5 — Run / Verify

- GPT запускает команду или тест
- видно успешный результат

### Shot 6 — Payoff

- финальный текст:
  - `The work kept moving`
  - `Second Lane finished the job inside ChatGPT`
