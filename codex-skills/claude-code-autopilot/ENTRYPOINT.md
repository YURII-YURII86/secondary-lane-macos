# Claude Code Autopilot — точка входа

Этот скилл устанавливает и разворачивает **Second Lane** на macOS
полностью автоматически через Claude Code.

## Что читать ПЕРВЫМ

1. **Этот файл** — ты уже читаешь
2. **SKILL.md** — главная инструкция, прочти перед любым действием
3. **references/01-state-machine.md** — таблица состояний, держи
   открытой во время всей работы

## Всё остальное — по требованию

| Файл | Когда читать |
|---|---|
| `references/02-verification-checklist.md` | Перед тем как сказать «готово» |
| `references/03-troubleshooting.md` | Когда что-то пошло не так |
| `references/04-ngrok-guide.md` | Шаг S3 — настройка ngrok |
| `references/05-chatgpt-builder.md` | Шаг S10 — сборка GPT |

## Адаптивное обнаружение проекта

Перед чем-либо — найди корень проекта:

```bash
python3 codex-skills/gpts-mac-autopilot/scripts/discover_secondarylane_layout.py . --max-depth 4
```

Используй `candidate_root` из `best_match` как основу для всех дальнейших путей.
Никогда не хардкодь `Версия для Мак/` или любое другое имя папки.

## Главное правило

Делай максимум сам. Останавливайся только там, где **физически** нужен
человек (логин в браузере, сканирование QR-кода, ввод кредитки).
Всё остальное — автоматически, без лишних вопросов.
