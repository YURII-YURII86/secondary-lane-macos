# Runbook: Clean Mac

Используй, если на Mac почти ничего не настроено.

Порядок:

1. найти проект
2. проверить `python3.13`
3. поставить Python 3.13
4. проверить `ngrok`
5. поставить `ngrok`
6. провести через ngrok login/token/domain
7. создать `.env`
8. запустить панель
9. проверить tunnel
10. собрать GPT
