# Node.js Projects

Основные признаки:

- есть `package.json`;
- есть lock-файл: `package-lock.json`, `pnpm-lock.yaml`, `yarn.lock`, `bun.lockb`;
- есть папки `src`, `app`, `pages`, `components`, `server`.

Что проверить в первую очередь:

- `scripts` в `package.json`;
- какой пакетный менеджер используется;
- есть ли `typescript`;
- есть ли `vite`, `next`, `react`, `express`, `nest`.

Полезные вопросы к проекту:

- как запускается локально;
- как собирается;
- есть ли тесты;
- где серверная часть, если она есть;
- есть ли отдельные env-файлы.

Если задача про починку:

1. Найди скрипты `build`, `dev`, `test`, `lint`.
2. Прочитай ошибку, а не угадывай.
3. Меняй только тот слой, где ошибка реально проявилась.
4. После правки снова запусти проверку.
